# QKD Parameter Optimization
## Overview
This project implements a research-oriented framework for parameter optimization in BB84 quantum key distribution (QKD) protocol. The goal is to investigate how classical optimization and machine learning techniques can accelerate parameter tuning in QKD systems. The project consists of following four stages:
1. BB84 Simulator
2. Local search algorithm based dataset generation
3. Machine learning modeling
4. (optional) Optimization
5. Finite-key analysis

The framework is designed as a small-scale research prototype inspired by wang (2019)
